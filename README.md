# Appointment Scheduler / Időpontfoglaló

## English

This is a simple **Appointment Scheduler** web application built using **HTML, CSS, and JavaScript**.  
It allows a user to select a day and a time for an appointment and displays a confirmation message.  

### Features
- Choose a day (Monday to Friday)  
- Choose a time (9:00 – 16:00)  
- Book appointment with confirmation popup  
- Simple fade-in animation for confirmation  
- Fully responsive design  
- Beginner-friendly code demonstrating DOM manipulation and event handling  

### How to use
1. Open `index.html` in your browser  
2. Select a day and a time from the dropdown menus  
3. Click the "Book Appointment" button  
4. See the confirmation popup  

### Technologies
- HTML  
- CSS  
- JavaScript

## Magyar

Ez egy egyszerű **Időpontfoglaló** webalkalmazás, **HTML, CSS és JavaScript** használatával készült.  
A felhasználó kiválaszthat egy napot és egy órát az időpontfoglaláshoz, és a rendszer egy visszaigazoló üzenetet jelenít meg.  

### Funkciók
- Nap kiválasztása (Hétfőtől Péntekig)  
- Óra kiválasztása (9:00 – 16:00)  
- Időpont foglalása visszaigazoló popup-pal  
- Egyszerű fade-in animáció a visszaigazoláshoz  
- Teljesen reszponzív design  
- Kezdőbarát kód, bemutatja a DOM manipulációt és az eseménykezelést  

### Használat
1. Nyisd meg az `index.html` fájlt a böngésződben  
2. Válassz napot és időpontot a legördülő menükből  
3. Kattints a „Book Appointment / Foglalás” gombra  
4. Megjelenik a visszaigazoló popup  

### Technológiák
- HTML  
- CSS  
- JavaScript
